## 🚀 Technology Stack

### Frontend
- **Framework:** Next.js 14+
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Library:** Shadcn/ui
- **State:** Zustand + React Query

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Real-time:** Socket.io
- **Auth:** JWT + bcrypt

### Database
- **Primary:** MongoDB (Atlas)
- **Caching:** Redis
- **Storage:** AWS S3 / Cloudinary

### Mobile
- **Framework:** React Native
- **Tools:** Expo

### Infrastructure
- **Hosting:** Hostinger VPS KVM4
- **Server:** Nginx + PM2
- **SSL:** Let's Encrypt
- **CDN:** Cloudflare
